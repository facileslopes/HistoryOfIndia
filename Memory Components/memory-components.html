RAM (Random Access Memory)

The RAM, also called read-write memory or direct access memory, is a volatile memory that can be read from or written to multiple times. The data will be lost when the computer system is switched off. Any byte of data stored can be accessed directly without affecting the other bytes. RAM stores information currently in use allowing the computer to quickly access this information when required. RAM can be divided into two types: Dynamic RAM(DRAM), and Static RAM(SRAM). DRAM is built out of memory cells, each with an electricity-leaking capacitor and a transistor, and requires charging regularly(every few milliseconds) so that the data is preserved. While DRAM requires periodic refreshing to store data, it is cheaper compared to SRAM and also has a higher data storage density, and is most frequently used in main memory. SRAM is able to store memory through flip-flops that can store data as long as the computer is switched on. They do not require refreshing, but do have faster access speeds compared to DRAM, and are used in cache memory.

ROM (Read Only Memory)

ROM, as the name implies, can only be read and not written to or edited. It shares the properties of random access with RAM, but unlike RAM, is non-volatile. ROM is primarily used for the storage of data or programs that are meant to be reused without change, such as bootstrap programs built to boot up a system when it is turned on. ROM has many other uses, such as in embedded systems, and can be divided into four types.

  1. The simplest type of ROM where all required data is installed into a chip as part of the manufacturing process. The chip must be replaced in order to accomodate any changes needed by the system.
  2. Programmable ROM(PROM) is a form of storage where each bit is locked by a fuse or antifuse. When developing PROMs, data is first sent to a system builder to test the programmable chip before it is committed to all of the provided chips. PROM cannot be written once installed.
  3. Erasable PROM(EPROM) is more flexible than PROM in that data can be erased from the chip by using a UV light, and new data can be written back into it, though it usually requires removing the chip from the device.
  4. Electrically Erasable PROM is the most flexible ROM type, where an electrical signal is used to remove existing data instead of UV light, and its contents can be changed without removing the chip from the device. However, EEPROM is still used like a regular ROM most of the time, as a read-only chip.

Buffers

A buffer is a temporary storage that is created for data being transmitted from one location in a system to another that functions as a queue. The queue holds data that is sent more quickly than it can be received. Buffers help ensure that data is transferred in the exact order that it entered the buffer.
